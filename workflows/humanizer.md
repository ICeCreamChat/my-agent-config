# 学术语气润色工作流

基于 **humanizer**（by [@blader](https://github.com/blader)）

## 技能路径

```
c:\Users\ICe\.agent\AcademicForge\skills\humanizer
```

## 核心能力

- ✍️ **学术语气优化** — 调整措辞使其符合学术写作规范
- 📖 **可读性提升** — 改善句式结构，增强逻辑连贯性
- 🛡️ **避免 AI 检测特征** — 减少机器生成文本的典型模式
- 📝 **同行评审准备** — 润色草稿使其达到投稿水准

## 使用方式

1. 进入技能路径，找到 `SKILL.md`
2. 阅读完整指令
3. 将用户的学术草稿按指令进行语气优化和润色

## 适用场景

- 论文终稿润色
- 摘要与引言的语气调整
- 学术邮件与审稿回复的语气优化
- 非母语作者的英文学术写作改善
